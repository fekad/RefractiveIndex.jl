RefractiveIndexDatabase.Cauchy — Type

Cauchy formula:

\[n = c_1 + \sum \limits_{i=5}^{N} c_{2i} \lambda^{c_{2i+1}}\]

RefractiveIndexDatabase.Exotic — Type

Exotic formula:

\[n^2 = c_1 + \frac{c_{2}}{\lambda^2 - c_{3}} + \frac{c_{4}(\lambda - c_{5})}{(\lambda - c_{5})^2 + c_{6}}\]

source
RefractiveIndexDatabase.Gases — Type

Gases formula:

\[n - 1 = c_1 + \sum \limits_{i=1}^{N} \frac{c_{2i}}{c_{2i+1} - \lambda^{-2} }\]

source
RefractiveIndexDatabase.Herzberger — Type

Herzberger formula:

\[n = c_1 + \frac{c_{2}}{\lambda^2 - 0.028} + c_{3} \left( \frac{1}{\lambda^2 - 0.028} \right)^2 + \sum \limits_{i=1}^{N} c_{i+3} \lambda^{2i}\]

source
RefractiveIndexDatabase.Polynomial — Type

source
RefractiveIndexDatabase.RIInfo — Type

RefractiveIndex.INFO formula:

\[n^2 = c_1 + \frac{c_{2} \lambda^{c_{3}}}{\lambda^2 - c_{4}^{c_{5}}} + \frac{c_{6} \lambda^{c_{7}}}{\lambda^2 - c_{8}^{c_{9}}} + \sum \limits_{i=5}^{N} c_{2i} \lambda^{c_{2i+1}}\]

source
RefractiveIndexDatabase.Retro — Type

Retro formula:

\[\frac{n^2 - 1}{n^2 +2} = c_1 + \frac{c_{2} \lambda^2}{\lambda^2 - c_{3}} + c_{4} \lambda^2\]
